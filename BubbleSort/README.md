# 冒泡排序
## 1.原理
依次比较两个相邻元素，如果前者大于后者将交换两者的值(相等不交换)
## 2.思路
    tip: n = array.size()
    (1). 比较array[0]与array[1]，小数放前面，大数放后面
    (2). 比较array[1]与array[2], 小数放前面，大数放后面
    ...
    (n-1). 比较array[n-3]与array[n-2], 小数放前面，大数放后面
    (n). 比较array[n-2]与array[n-1], 小数放前面，大数放后面
